# Private Company Diligence Collection v1

## Purpose

Track locally collected official-company and financing materials for
private-company follow-up.

## Inputs

1. `private_company_diligence_plan_batch`
2. local document manifest

## Output

One collection per private-company entity with:

1. collected documents
2. local file existence
3. document counts

## Boundary

This is not a filing collection object.
It is the private-company analogue used for official company materials and
financing sources.
