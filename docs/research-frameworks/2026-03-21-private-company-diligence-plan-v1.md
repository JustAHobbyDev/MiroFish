# Private Company Diligence Plan v1

## Purpose

Route real bounded-lane entities that resolve to private-company status away
from public filing collection and into a deterministic diligence plan.

## Inputs

1. live issuer-resolution results

## Output

One plan per private-company entity with:

1. route type
2. source priorities
3. query terms
4. next actions

## Boundary

This object is not a filing-collection substitute.
It is a routing and planning object for private-company follow-up.
