# Utility Lane Review Surface v1

## Purpose

Give the utility and large-load power lane one consolidated downstream review
object.

## Inputs

1. `utility_lane_downstream_state`
2. utility follow-up queue
3. filing support
4. private-company diligence collection

## Output

One row per entity with:

1. route type
2. evidence counts
3. next action
4. public-filing or private-company disposition
